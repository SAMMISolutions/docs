Remaps a number from one range into another range.

For example, remapping `50` from the input range `0` to `100` into the output range `0` to `1` returns `0.5`.

| Box Name | Type | Description | 
|-------|--------|--------
|Number|Number {% include asterisk.html%}|Value to remap.
|Input Minimum|Number {% include asterisk.html%}|Minimum value of the original range.
|Input Maximum|Number {% include asterisk.html%}|Maximum value of the original range.
|Output Minimum|Number {% include asterisk.html%}|Minimum value of the target range.
|Output Maximum|Number {% include asterisk.html%}|Maximum value of the target range.
|Save Variable As|String|Variable name to save the result.
